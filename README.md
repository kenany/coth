# coth

Calculate the hyperbolic cotangent of a number.

## Example

``` javascript
const coth = require('coth');

coth(Math.PI);
// => 1.0037418731973
```

## Installation

``` bash
$ npm install coth
```

## API

``` javascript
const coth = require('coth');
```

### `coth(x)`

Given a _Number_ `x`, returns the hyperbolic cotangent of `x` as a _Number_`.
